## ✨ 功能特色

### 🎲 戰棋海克斯選擇系統
- **高度擬真**：從 **10 個專屬海克斯** 中隨機抽取 **3 個** 進行初始展示。
- **重新整理機制**：每張卡片皆備有獨立的「刷新」按鈕，可進行 **1 次獨立刷新**，並配有防重複連點與防止池空崩壞的指針阻斷保護（`pointer-events: none`）。
- **確認與鎖定**：選定海克斯後按下鎖定，鎖定後卡片亮起、其餘變暗，並於最後的星球頁左側展示您選定的海克斯。

### 🔊 互動音效控制系統
- **全站音效反饋**：刷新卡片、鎖定海克斯、打開神秘禮物、翻開今日運勢以及切換頁面，皆配有音效。
- **智慧音樂消長 (Ducking Effect)**：在吹蠟燭等關鍵音效播放時，背景音樂會智慧調小音量，播放結束後自動恢復。
- **喇叭控制按鈕**：首頁設有小喇叭，可隨時手動開關音效，且在音效播放完畢後自動重設按鈕狀態。

### 🎂 3D 蛋糕許願區
- **動態蠟燭火焰**：CSS 蠟燭火焰動畫，並配有對話框。
- **吹蠟燭許願**：點擊蛋糕觸發 **3 秒許願倒數**，火焰熄滅，隨即噴發 **粒子特效**，搭配音效，營造極具儀式感的慶生氛圍。

### 🪐 互動式好友星軌祝福系統
- **動態橢圓星軌**：中央星系動態計算正弦與餘弦公式，讓 **4 位摯友星球** 沿著各自精準的橢圓軌道實時旋轉。
- **好友在線名單**：右側面板展示好友在線列表，備有動態在線綠燈，所有好友與星座羈絆皆支援 **Hover Tooltip 浮動資訊框**。
- **專屬祝福語音**：點擊好友星球，會彈出祝福卡片，並**播放該位好友錄製的生日快樂祝福語音**！

### 🔮 3D 運勢抽籤系統
- **3D 翻牌動畫**：採用 CSS 3D 空間翻轉特效（`backface-visibility` 與 `transform-style: preserve-3d`），模擬真實翻牌手感。
- **吉不吉運勢**：隨機抽取大吉、中吉、小吉、平順或神秘運勢，點擊後翻開，每次重返首頁時自動重置。

### 🎁 神秘禮物盒
- **開箱驚喜**：點擊禮物盒觸發驚喜音效，彈出卡片，展示 10 種隨機禮物之一的描述與表情包。
- **狀態保存**：打開後禮物盒會變暗封存，關閉彈窗後禮物盒轉換為表情包展示。

### ⏰ 生日倒數與歡慶霓虹彩盒
- **精準計時鎖定**：右上角精準倒數距離 **8 月 9 日** 還有多久，並在生日當天添加霓虹特效。
- **鎖定日期測試**：自動偵測真實生日（8/9），可在測試時切換鎖定 / 非鎖定模式。
- **視覺發光升級**：生日當天，計時框會變身為粉、紫、金三色交織的**霓虹呼吸框**。

### 📱 行動裝置警告防禦
- **佈局保護**：檢測寬度小於 1000px 的螢幕時，會自動切換為行動版防禦警告，提示「請使用電腦開啟」，以保護遊戲客戶端佈局不被手機窄螢幕擠壓。

## 🛠️ 亮點與優化

1. **動態等比例自適應縮放（Viewport Auto-Scaling）**：
   - 傳統 CSS 響應式佈局在面對「帶有精準橢圓軌道座標、大量絕對定位、CSS 3D 翻牌」等互動遊戲畫面時，極易因為瀏覽器視窗縮小而導致佈局崩潰或重疊。
   - 本專案採用 **JS 動態縮放技術** (`resize.js`)，自動將 1920px 設為設計基準，依據當前視窗寬度動態計算並為 `document.body` 賦予 `zoom` 縮放值（安全區間限制在 `0.5` ~ `1.5`），完美實現了在任何解析度縮放比（如 125%、150%）下，網頁版面皆保持 100% 相同比例的質感！
   
2. **分頁切換暫停與垃圾回收機制（Visibility Shield）**：
   - 解決了傳統定時器在背景分頁執行的經典堆積 Bug。在 `galaxy.js` 中加入了對 `visibilitychange` 事件的實時監聽，當用戶切換到其他分頁時，會**立刻清除彈幕定時器並清空舞台 DOM 元素**，防止瀏覽器在背景產生排隊 backlog；當用戶切換回來時，則自動重新初始化，保證彈幕永遠均勻流暢！

3. **完美對稱置中面板設計**：
   - 重新調校了星球頁面（`#pg4`）的三欄式布局。透過移除不對稱的 paddings，並在 `.screen` 容器內套用 `justify-content: center` 垂直置中，實現了左右面板、中央星系在任何筆電高度下，上下間距皆保持數學上 1:1 的完美對稱，帶來極致平衡的視覺舒適感。

4. **圖層疊加與 Z-Index 重構**：
   - 重新梳理了全站的圖層關係（Z-Index）。將通用彈窗遮罩 `.modal` 的 `z-index` 全域調升至 `9999`，徹底解決了原本大螢幕下右上角音樂鈕或倒數計時器在彈窗顯示時仍浮在最上層的層次錯亂，確保覆蓋層無懈可擊。

5. **海克斯卡片池安全邊界保護**：
   - 強化了海克斯刷新系統的邊界防禦。透過 `remainingPool.length > 0` 驗證與 `pointer-events: none` 實體阻斷，徹底根治了玩家連點「刷新」按鈕導致卡片資料變成 `undefined`、卡片消失以及控制按鈕異常的重大 Bug。
